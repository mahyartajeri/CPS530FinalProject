How to create a web page using Semantic UI and Ruby on Rails


Semantic UI

Text

Text can be colored or resized using the text class. Note: Only usable within <span> tags

Colour

Syntax goes in the format: <span class="ui colour text">The text you want formatted</span>

Ex: This is <span class="ui red text">red</span> text and this is <span class="ui blue text">blue</span> text and this is <span class="ui purple text">purple</span> text

The code above has the following output:

This is red text and this is blue text and this is purple text



Size

Syntax goes in the format: <span class="ui size text">The text you want formatted</span>

Ex: This is <span class="ui small text">small text</span>

This is <span class="ui medium text">medium text</span>

This is <span class="ui large text">large text</span>

The code above has the following output:


This is small text

This is medium text

This is large text

Images

There are a wide variety of image class modifications

Size

For example, you can easily edit how big you want your images to be with the size modification. Much like you do with text.
Syntax goes in the format: <img class="ui size image" src="picture url or path">

Ex: <img class="ui small image" src="https://upload.wikimedia.org/wikipedia/commons/thumb/b/bf/African_Elephant_%28Loxodonta_africana%29_male_%2817289351322%29.jpg/330px-African_Elephant_%28Loxodonta_africana%29_male_%2817289351322%29.jpg">

Ruby on Rails

MVC Architecture

Rails is made of a Model-View-Control architecture:

Model is your database

View is your web page

Control is the backend code of your web application

Creating your first page

If you go to the Ruby directory you made on the previous page, you'll find a variety of sub-directories. The one we'll be focusing on is the app directory.

Creating a web page involes initializing a controller and a view, you can do so by running the following command:

cd Ruby

This puts you into the directory you made on the previous page, next run:

rails generate controller home index

View

The command above created a new view in the views sub-directory of apps. You can access the view in the app/views/home/index.html.erb file.

From now on, you can create your webpage in regular html from index.html.erb . You can access it via url using http://localhost:3000/home/index
